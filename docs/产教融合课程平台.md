## 1. 产品定位与边界

### 1.1 定位
面向产教融合的**学分教学一体化平台**，按三个独立子平台划分职责：

| 平台 | 核心职责 | 使用者 |
|------|---------|--------|
| **课程管理平台** | 课程资源的精细化编辑、审批、发布；题库/试卷/作业模板管理；资源中心维护 | 管理员、教研组长、主讲教师 |
| **学生学习平台** | 前台课程浏览、自主学习（视频/阅读/课件）、课后作业提交、测验答题、成绩查看 | 学生 |
| **智慧课堂平台** | 接收教务系统课表、课前备课（导学/测验/签到配置）、课中授课（课件放映/互动/签到）、课后学情分析 | 教研组长、主讲教师 |

### 1.2 系统边界

| 功能 | 本平台 | 教务系统 |
|------|--------|---------|
| 课程资源编辑（体系课/颗粒课） | ✅ 课程管理平台 | ❌ |
| 题库/试卷/作业模板管理 | ✅ 课程管理平台 | ❌ |
| 学生自主学习（视频/阅读/作业/测验） | ✅ 学生学习平台 | ❌ |
| 成绩录入/查看/发布 | ✅ 学生学习平台+智慧课堂 | ❌ |
| 教师备课/上课/课堂互动 | ✅ 智慧课堂平台 | ❌ |
| **排班排课** | ❌ | ✅ |
| **学籍/班级管理** | ❌ | ✅ |
| **学分计算/毕业审核** | ❌ | ✅ |
| **实践成绩汇集** | ❌ | ✅ 教务系统从场景化平台汇集后，本平台按需接收 |

**数据流向**：
- 教务系统 → 本平台：学期课表、学生名单、课程基础信息、实践成绩（如需）
- 本平台 → 教务系统：过程性成绩明细、期末总评成绩、出勤数据、课堂互动数据

## 2. 用户角色与权限

### 2.1 角色定义

| 角色 | 核心诉求 |
|------|----------|
| **平台管理员** | 审批流配置、用户账号管理、运营配置（Banner/推荐）、公共题库/资源库维护、全局数据监控 |
| **教研组长 / 系主任** | 审批课程质量、配置审批流程与批次分组、教学督导、成绩复核审批、查看本系学情 |
| **主讲教师** | 创建/编辑课程、按课表备课与授课、下发测验/作业、录入与提交成绩、查看所授课程学情 |
| **学生** | 浏览课程、在线学习、完成测验与作业、查看成绩与反馈、参与课堂互动 |

> **说明**：原"协同教师"角色已合并至主讲教师。主讲教师可邀请其他教师以"只读"方式查看课程与学情，但操作权限与主讲教师一致，不再单独设立角色。

### 2.2 权限矩阵

| 功能 | 管理员 | 教研组长 | 主讲教师 | 学生 |
|------|--------|---------|---------|------|
| 创建/编辑课程 | ✅ | ✅ | ✅ | ❌ |
| 删除课程 | ✅ | ✅ | 仅自己的 | ❌ |
| 提交/审批课程 | 提交❌审批❌ | 提交✅审批✅ | 提交✅审批❌ | ❌ |
| 上下课/下发测验/课堂互动 | ❌ | ❌ | ✅ | ❌ |
| 查看学情 | 全局 | 全局 | 自己的课 | 仅自己 |
| 学习/答题/参与课堂 | ❌ | ❌ | ❌ | ✅ |
| 课程收藏 | ❌ | ❌ | ❌ | ✅ |
| 成绩录入/提交成绩册 | ❌ | ❌ | ✅ | ❌ |
| 成绩审核/发布/修改审批 | ❌ | ✅ | ❌ | ❌ |
| 成绩查看 | 全局 | 全局 | 自己的课 | 仅自己 |
| 配置审批流程/批次 | ✅ | ✅ | ❌ | ❌ |

### 2.3 数据可见范围

课程管理平台内所有课程/资源列表按三态隔离：
- **我的**：仅本人创建
- **共建**：被邀请协作的教师可见（只读查看，不可编辑）
- **公共库**：已发布且标记为公共，全员可见

## 3. 平台一：课程管理平台（教材制作）

> 本平台负责课程从创建到发布的全生命周期管理，以及教学测评素材（题库、试卷、作业模板）的维护。所有课程内容在此编辑完成后发布，供学生学习平台和智慧课堂平台调用。

---

### 3.1 课程资源体系

#### 3.1.1 课程形态

| 形态 | 结构特点 | 适用场景 |
|------|---------|----------|
| **体系课** | 树形节点（最多三级：章→节→课时） | 系统化专业技能传授、完整学期课程 |
| **颗粒课** | 扁平结构，单课时/单知识点 | 碎片化技能点，可被体系课节点引用或克隆 |

#### 3.1.2 分类维度

- **按行业**：电子信息、软件测试、公共管理与服务、计算机、交通运输等 14+ 行业枚举
- **按专业**：软件工程、自动化、人工智能、大数据管理等
- **按课程属性**：公共基础课、专业基础课、专业核心课、素质拓展课
- **按状态**：未提交（草稿）、审批中、已驳回、已发布

#### 3.1.3 体系课节点类型

| 类型 | 标签色 | 说明 | 编辑权限 |
|------|--------|------|---------|
| **原创** | 紫色 | 从零编排的空白节点 | 完全可编辑 |
| **克隆** | 橙色 | 复制已有颗粒课内容，生成独立副本 | 完全可编辑，与源颗粒课解耦 |
| **引用** | 蓝色 | 关联已有颗粒课，随源内容同步更新 | 只读展示，不可编辑 |
| **资源节点** | 绿色 | 仅作资源挂载点，无独立教学内容 | 仅名称和资源可编辑 |

#### 3.1.4 课程生命周期状态机

```
                     +-----------+
                     |   创建    |
                     |  (草稿)   |
                     +----+------+
                          |
                          v
               +---------------------+
               |      编辑内容        |
               |  (可反复保存草稿)     |
               +----------+----------+
                          |
               +----------+----------+
               |                     |
               v                     v
        +-------------+       +-------------+
        |   提交审批   |       |   直接放弃   |
        +------+------+       +-------------+
               |
               v
        +-------------+
        |   审批中    |
        +------+------+
               |
       +-------+-------+
       |               |
       v               v
 +----------+    +----------+
 | 审批通过  |    | 审批驳回  |
 | (已发布) |    | (已驳回)  |
 +----+-----+    +----+-----+
      |               |
      v               v
 +----------+   +-------------+
 | 新版本   |   |  修改后重提  |
 | (草稿)   |   +------+------+
 +----+-----+          |
      |                +-----> 提交审批（循环）
      +-----> 提交审批（简化审批流）
```

> **版本升级规则**：已发布课程若需修改，创建新版本草稿，走**简化审批流**（仅教研组长快速审核变更差异，高亮对比），无需全量重新审批。

#### 【本节关键业务规则】

1. **层级限制**：体系课目录树最多三级（章→节→课时），无限层级会导致前端渲染和导航体验复杂，三级已覆盖绝大多数教学场景。
2. **节点引用规则**：引用型节点随源颗粒课更新而同步，不可独立修改；克隆型节点为独立副本，与源颗粒课解耦，可独立修改。
3. **版本管理**：已发布课程修订时创建新版本草稿，走简化审批流；审批通过后旧版本自动归档，前台仅展示最新版本。
4. **数据权限隔离**：后台按"我的 / 共建 / 公共库"做数据权限隔离；前台仅展示已发布课程。

---

### 3.2 体系课管理

#### 3.2.1 通用列表页规范

**顶部统计看板**：总数 / 未提交 / 审批中 / 已驳回 / 已发布（5 个统计卡片，点击可筛选）

**Tab 切换**：我的 / 共建 / 公共库

**筛选维度**：
- 关键字（课程名称/编码模糊搜索）
- 所属行业（多选下拉）
- 所属专业（多选下拉）
- 课程状态（单选）
- 批次分组（体系课专属，多选）
- 创建时间范围（日期选择器）

**视图模式**：列表视图（表格，支持排序）、卡片视图（封面+关键信息）、按状态分组视图，三种可切换，记忆用户上次选择

**批量操作**：
- 导入课程包（ZIP 格式，含课程结构+资源文件）
- 导出选中课程（Excel，含基础信息+状态+创建人）
- 批量删除（二次确认，仅草稿/已驳回状态可删）
- 批量提交审批（仅草稿状态）
- 批量取消审批（仅审批中状态，创建人可操作）
- 批量发布/取消发布（管理员/教研组长权限）

**单课操作**：查看 / 编辑 / 导出 / 邀请共建（生成只读链接）/ 复制（生成副本）/ 删除

**快捷操作栏**：配置审批流程、配置批次分组、新增课程（右侧抽屉快速创建，填写名称+行业+专业+属性后进入编辑页）

#### 3.2.2 体系课编辑页（三栏结构）

```
+------------------------------------------------------------------+
|  顶部标题栏：课程名称 + 状态标签 + 返回 / 暂存草稿 / 提交审批      |
+------------------+---------------------------+-------------------+
|   左侧：课程目录树 |        中间：节点编辑区      |   右侧：发布检查   |
|   （最多三级）     |   （根据节点类型变化）       |   （完整性进度）    |
+------------------+---------------------------+-------------------+
```

##### 课程基本信息（可折叠面板）

| 字段 | 必填 | 说明 |
|------|------|------|
| 课程名称 | 是 | 体系课全称，全局唯一校验 |
| 课程编号 | 是 | 支持系统按规则自动生成（行业码+专业码+序号），或手动填写 |
| 所属行业 | 是 | 下拉单选，枚举值由管理员维护 |
| 所属专业 | 是 | 级联下拉，先选行业后选专业 |
| 课程属性 | 是 | 公共基础课 / 专业基础课 / 专业核心课 / 素质拓展课 |
| 授课教师 | 是 | 主讲教师，默认当前登录人，可切换（需有教师权限） |
| 课程封面 | 否 | 建议 16:9，支持 JPG/PNG，最大 5MB |
| 课程简介 | 否 | 富文本编辑器，支持图文混排 |
| 课程状态 | — | 只读：未提交 / 审批中 / 已驳回 / 已发布 |

> **交互逻辑**：首次进入编辑页时，若课程名称未填写，自动展开基础信息面板并聚焦名称输入框；填写完成后方可进入节点编辑。基础信息变更后自动保存。

##### 左侧：课程目录树

- **层级结构**：最多三级（章→节→课时），以树形组件展示
- **节点展示**：序号（自动重算）+ 节点名称 + 类型标签（原创/克隆/引用/资源节点，带颜色区分）
- **拖拽排序**：同级节点支持拖拽调整顺序，释放后自动重算全树序号（如 1.1→1.2→1.3）
- **节点操作**：
  - 编辑名称：单击名称进入行内编辑，失焦保存
  - 添加子节点：点击「+」按钮，弹出配置抽屉（节点名称、上级节点、显示顺序）
  - 删除节点：点击删除图标，弹出确认弹窗（警告：子节点将一并删除），确认后级联删除
- **底部操作**：「+ 添加节点」按钮，默认添加同级节点；若选中节点为二级，则添加三级；三级节点无添加子节点按钮
- **选中态**：点击节点后高亮，中间编辑区加载对应节点内容

##### 中间：节点编辑区

**首次设置内容类型时**，展示选择面板（四选一）：
- 📁 **上传资源** → 变为「资源节点」，仅保留名称和资源上传功能
- 📚 **新增颗粒课** → 变为「原创」节点，加载完整编辑表单
- 🔗 **克隆颗粒课** → 弹出颗粒课选择弹窗，复制内容生成独立副本，名称自动加 "-克隆" 后缀
- 📎 **引用颗粒课** → 弹出颗粒课选择弹窗，建立只读关联，名称保持原名称

**原创/克隆节点可编辑内容：**

| 模块 | 字段/功能 | 详细说明 |
|------|----------|---------|
| **基本信息** | 节点名称 | 与目录树同步，修改后实时回写树节点 |
| | 学习目标 | 富文本编辑器，带简易 Markdown 工具栏（标题/列表/加粗/链接），至少填写 1 条 |
| | 涉及知识点 | 标签输入框，回车添加标签；输入时自动下拉建议前 6 个匹配项（来自知识图谱） |
| | 预估课时 | 数字输入，单位"课时"，必须 > 0 |
| **知识点标签** | 命中知识图谱 | 显示为 🔗 靛蓝色标签，参与图谱推送与关联课程推荐 |
| | 未命中知识图谱 | 显示为 📝 灰色标签，纯文本展示，不参与图谱推送 |
| **课程资源** | 文件上传 | 支持 PDF/PPT/DOC/XLS/MP4/ZIP/CSV，单文件最大 500MB，支持分片上传 |
| | 资源列表 | 列表展示：文件图标 + 文件名 + 大小 + 上传时间 + 类别（主讲课件/辅助课件） |
| | 操作 | 点击上传/拖拽上传；可移除、可替换；主讲课件标记为默认打开 |
| **测验配置** | 添加测验 | 点击「+ 添加测验」弹出配置抽屉 |
| | 试卷模式 | 从试卷库选择已有试卷，配置：考试时长、允许重考次数、题目乱序、交卷后显示成绩/答案、启用条件（随时作答/定时启用/后台手动启用） |
| | 题库模式 | Tab 切换（我的/共建/公共题库），勾选题目，配置抽题规则：自动抽题开关、抽题数量、难度分布（易/中/难比例） |
| **作业配置** | 添加作业 | 点击「+ 添加作业」弹出配置抽屉 |
| | 作业题目 | 富文本输入 |
| | 作业要求 | 多行文本，详细描述作答要求 |
| | 附件要求 | 开关：是否需要学生上传附件 |
| | 截止时间 | 仅当"需要附件"开启时展示日期时间选择器 |

**引用节点**：
- 全部字段只读展示，所有操作按钮禁用
- 底部提示横幅：「当前节点为引用模式，内容不可编辑；内容随源颗粒课实时同步更新；如需修改请删除后重新克隆」
- 展示源颗粒课信息：名称、编码、创建人、最后更新时间

**资源节点**：
- 仅展示节点名称（可编辑，同步回写目录树）+ 资源上传卡片
- 无学习目标、知识点、测验、作业模块

##### 右侧：发布检查面板

实时检查当前节点完整性，动态更新进度：

| 检查项 | 完成标准 | 未完成提示 | 权重 |
|--------|---------|-----------|------|
| 节点名称 | 已填写且非空 | 未设置节点名称 | 必检 |
| 学习目标 | 已填写至少 1 条 | 未设置学习目标 | 必检 |
| 涉及知识点 | 已关联 ≥1 个标签 | 未设置涉及知识点 | 必检 |
| 预估课时 | 数值 > 0 | 未设置预估课时 | 必检 |
| 课程资源 | 已上传 ≥1 个文件 | 未上传课程资源 | 必检 |
| 测验 | 已配置 ≥1 道题目（试卷或题库抽题≥1） | 未配置测验 | 必检 |

> 资源节点精简为 2 项：节点名称 + 课程资源

**进度展示**：完成项数 / 总项数 + 百分比进度条
- 全部完成：进度条绿色，提示"💡 所有检查项已完成，可以提交审批"
- 未完成：进度条琥珀色，提示"💡 建议完善 {字段名}，提升课程规划准确性"

##### 颗粒课选择弹窗（克隆/引用时弹出）

- **搜索区**：节点名称模糊搜索、课程编码精确搜索、创建人下拉选择
- **筛选**：行业、专业、状态（已发布）
- **表格**：单选表格，列：节点名称、编码、创建人、创建时间、状态、知识点数、课时、资源数
- **分页**：每页 10/20/50 条，支持跳转
- **操作**：取消 / 确定（确定后关闭弹窗，加载节点内容）

##### 保存与切换策略

- **自动暂存**：节点内容变更后 3 秒无操作自动本地暂存（LocalStorage），防止意外刷新丢失；恢复时提示"检测到未保存的本地暂存，是否恢复？"
- **手动保存**：点击「暂存草稿」保存为草稿状态，全局提示保存成功
- **提交审批**：点击「保存并提交审批」，先执行全局保存，再校验所有节点完整性，通过后进入审批中状态；若校验不通过，提示"存在 {N} 个节点未完成发布检查，请完善后提交"
- **节点切换提示**：若当前节点有未保存修改，点击其他节点时弹出确认："当前节点有未保存修改，是否保存？"（保存并切换 / 不保存切换 / 取消）
- **离开页面提示**：浏览器关闭/刷新时，若有未保存内容，触发 beforeunload 提示

#### 3.2.3 体系课详情页（管理端）

- **基本信息卡片**：展示课程名称、编号、行业、专业、属性、教师、状态、版本号、创建人、创建时间、更新时间
- **统计信息**：节点总数、原创节点数、克隆节点数、引用节点数、资源节点数、总课时、资源总数
- **审批记录时间轴**：纵向时间轴展示所有审批操作（提交/通过/驳回/重提），含审批人、时间、备注/原因
- **版本历史**：列表展示所有历史版本（版本号、创建时间、变更摘要、操作人），支持对比查看差异
- **操作按钮**：编辑（草稿/已驳回状态）、提交审批、删除（仅自己的草稿/已驳回）、导出课程包

#### 【本节关键业务规则】

5. **发布检查规则**：课程名称、编码、学习目标、知识点、课程资源、随堂测验为必填项，全部完成后才允许提交审批。提交审批时系统遍历所有节点，任一节点未完成则拦截并列出节点清单。
6. **自动暂存规则**：节点内容变更后自动本地暂存，防止意外刷新丢失；支持手动保存为草稿；离开页面前校验未保存状态。
7. **审批留痕规则**：所有审批操作（提交/通过/驳回/重提）永久记录审批人、时间、备注/原因，不可删除，以时间轴形式展示。
8. **版本归档规则**：新版本审批通过后，旧版本自动进入归档状态，不再前台展示，但可在管理端历史版本中查看与对比。

### 3.4 审批管理

#### 3.4.1 审批工作台

**入口**：教研组长/管理员专属菜单"课程审批"

**双 Tab**：待审批 / 已审批

**待审批列表**：
- 列：课程名称、编码、版本号、类型（体系课/颗粒课）、行业/专业、批次分组（体系课）、创建人、提交日期、剩余待审时长（提交后 72 小时标黄，120 小时标红）
- 排序：默认按提交时间倒序，支持按剩余时长正序
- 批量操作：批量通过（需填写统一备注）、批量驳回（需填写统一原因）

**审批操作**：
- **查看详情**：点击课程名称，打开详情抽屉（左侧展示课程结构树，右侧展示当前节点内容与发布检查状态）
- **通过**：弹出确认框，填写审批备注（可选，默认"审批通过"），确认后课程状态变为"已发布"，创建人收到通知
- **驳回**：弹出表单，必须填写驳回原因（至少 10 字符），可选择性勾选"需修改项"（多选：基础信息/目录结构/节点内容/测验/作业），确认后状态变为"已驳回"，创建人收到通知并可在编辑页顶部看到驳回横幅

**已审批列表**：
- 列：课程名称、编码、版本、审批结果、审批人、审批时间、备注/原因
- 筛选：按结果（通过/驳回）、按时间范围、按审批人
- 操作：查看详情（只读）、查看审批记录时间轴

#### 【本节关键业务规则】

12. **审批时效**：教研组长应在提交后 5 个工作日内完成审批；超期未审系统自动提醒教研组长及管理员。
13. **驳回后重提**：已驳回课程修改后再次提交，走正常审批流（非简化流），确保修改质量。
14. **审批不可撤回**：审批操作（通过/驳回）一旦提交不可撤回或修改，避免课程状态混乱。

### 3.5 批次分组与审批流程配置

#### 3.5.1 批次分组管理

**批次列表**：
- 列：分组名称、批次编号（自动生成，格式：B+年月+序号）、所属院系、关联审批流程、体系课数量、创建时间、操作
- 操作：编辑 / 查看关联课程 / 删除（无关联课程时可用）

**新建/编辑批次**：
- 分组名称：必填，同一院系下不可重复
- 所属院系：下拉选择（数据来自教务系统或本系统维护）
- 关联审批流程：从流程模板单选，必填
- 批次编号：系统自动生成，不可编辑

**规则**：同一批次下所有体系课共用同一套审批流程；颗粒课不绑定批次。

#### 3.5.2 审批流程配置

**流程列表**：
- 列：流程名称、说明、审批步骤（以标签顺序展示：教研组长 → 系主任 → 管理员）、创建时间、绑定批次数、状态（启用/停用）
- 操作：编辑 / 复制 / 删除（无绑定批次时）/ 启用/停用

**新建/编辑流程**：
- 流程名称：必填
- 说明：多行文本
- 审批步骤：动态添加，每步包含：
  - 步骤名称（如"教研组长初审"）
  - 审批角色（下拉：教研组长/系主任/管理员，支持多选"任一具备角色的人均可审批"）
  - 审批方式（会签/或签，多审批人时）
- 步骤排序：拖拽调整顺序，至少保留一步
- 模板类型：单级审批（一步）/ 多级审批（两步及以上）

**删除限制**：存在绑定批次的流程不可删除，需先解绑或删除批次。

#### 【本节关键业务规则】

15. **批次绑定强制**：体系课创建时必须选择批次分组（已创建课程可在编辑页修改批次），未选择批次不可提交审批。
16. **流程变更影响**：审批流程修改后，已绑定该流程的批次即时生效；处于"审批中"状态的课程仍按旧流程完成当前审批，新提交课程按新流程执行。
17. **流程停用**：停用的流程不可被新批次选择，已绑定批次可继续使用直至批次失效。

### 3.6 测评中心

#### 3.6.1 题库管理

**Tab 切换**：我的 / 共建 / 公共题库（公共题库仅管理员可维护）

**题目列表**：
- 列：题目名称（题干前 30 字）、类型（单选/多选/判断/简答）、难度（易/中/难，颜色区分）、所属题库、知识点标签、创建人、创建时间
- 筛选：类型、难度、知识点、创建时间
- 批量操作：批量删除、批量移动至题库、批量导出（Word/Excel）

**题目新增/编辑**：
- **题干**：富文本，支持插入公式（LaTeX）、图片
- **题目类型**：单选 / 多选 / 判断 / 简答
  - 单选/多选：动态添加选项（最少 2 项，最多 10 项），设置正确答案（单选单选，多选多选）
  - 判断：仅"正确"/"错误"两个选项
  - 简答：参考答案文本框
- **选项**：每项支持富文本，可插入图片
- **正确答案**：根据类型自动校验（如多选至少选 2 项）
- **解析**：富文本，学生答题后展示
- **分值**：数字，默认 1 分
- **难度**：单选（易/中/难）
- **知识点标签**：同课程知识点标签输入，命中图谱显示 🔗

**题目预览**：编辑时实时预览学生端展示效果（题干+选项布局）

#### 3.6.2 试卷管理

**试卷列表**：
- 列：试卷名称、题数、总分、创建人、状态（草稿/已发布）、创建时间
- 操作：编辑 / 预览 / 复制 / 删除

**组卷方式**：
- **手动组卷**：从题库逐题选择，支持按知识点/难度/类型筛选；已选题目列表支持拖拽排序、删除、修改分值
- **自动组卷**：配置规则后系统抽题
  - 抽题范围：我的/共建/公共题库（多选）
  - 题目数量：总题数
  - 难度分布：易 _%、中 _%、难 _%（自动校验总和 100%）
  - 题型分布：单选 _ 题、多选 _ 题、判断 _ 题、简答 _ 题
  - 知识点范围：可选限定特定知识点（不选则为全部）
  - 生成后支持人工微调：替换题目、调整顺序、修改分值

**考卷设置**（试卷级配置，应用于所有引用该试卷的测验）：
- 考试时长：分钟，0 表示不限时
- 允许重考：开关（开则记录最高/最近/平均成绩，可配置）
- 题目乱序：开关（每位学生题目顺序随机）
- 选项乱序：开关（选择题选项顺序随机）
- 交卷后显示成绩：开关
- 交卷后显示答案与解析：开关（依赖"显示成绩"开启）
- 启用条件：随时作答 / 定时启用（设置开始/结束时间）/ 后台手动启用

**试卷预览**：完整模拟学生端答题界面，支持翻页、标记、提交

#### 3.6.3 作业模板管理

**模板列表**：
- 列：模板名称、适用课程数、创建人、创建时间
- 操作：编辑 / 应用至课程 / 复制 / 删除

**模板内容**：
- 模板名称：必填
- 作业题目：富文本
- 作业要求：多行文本，详细描述作答规范、评分标准
- 是否需要附件：开关
- 建议截止时间：相对时间（如"发布后 7 天"）或绝对时间
- 是否启用互评：开关（开启后配置互评权重、互评份数）

**应用至课程**：选择目标课程（我的/共建），一键生成作业并挂载到课程节点。

#### 【本节关键业务规则】

18. **题库权限**：教师仅可查看/编辑"我的"题库和"共建"题库；"公共题库"仅管理员可维护，所有教师可引用。
19. **试卷引用保护**：已发布的试卷被课程引用后，编辑试卷会提示"该试卷已被 {N} 门课程引用，修改将影响所有引用课程；建议创建新版本"。若强制修改，引用课程中的测验同步更新。
20. **题目唯一性**：同一题库内题干（纯文本去重）不可重复，防止题库膨胀。
21. **自动组卷校验**：自动组卷时若题库中符合条件的题目数量不足，提示"符合条件的题目仅 {N} 道，不足要求 {M} 道，请放宽条件或补充题库"。

#### 4.2.1 整体布局

**顶部导航**：
- Logo 点击返回首页
- 课程名称点击返回课程详情页
- 学习总进度条：全局进度（已完成章节数/总章节数），悬停显示"已完成 X/Y 章"
- 用户信息：头像+姓名下拉（个人中心/退出）

#### 4.2.2 左侧章节导航

- **宽度切换**：窄版（仅显示完成状态图标，宽度 60px）/ 宽版（完整信息，宽度 280px），点击边缘按钮切换，记忆用户偏好
- **头部信息**：课程名称（两行省略）、课程标签、总章/节数、总学习进度条（可视化）
- **章节树**：
  - 章级：可折叠，显示章标题 + 本章进度条（已完成节数/总节数）
  - 节级：显示状态图标（🔘 未开始 / 🟡 进行中 / ✅ 已完成）+ 类型图标（▶️ 视频 / 📄 阅读 / 📝 测验 / 📤 作业）+ 节标题 + 时长
  - 当前学习节：高亮背景，左侧带品牌色竖线指示器
  - 点击节标题：右侧加载对应内容
- **折叠动画**：宽版↔窄版切换时带平滑过渡动画

#### 4.2.3 右侧内容区 - 四种视图

*1. 章首页（点击章标题时展示）*
- 章节大标题 + 章节描述
- 本章学习目标卡片（带勾选框，学习完成后自动勾选）
- 本章知识点标签云
- 本章测验入口按钮（显示历史最高分，未做过显示"开始测验"）
- 本章作业入口（显示提交状态）
- 本章学习进度：X/Y 节已完成，进度环可视化

*2. 小节学习页（视频/阅读/课件）*
- **主区域**：
  - 视频类型：视频播放器组件，支持播放/暂停、进度拖拽、倍速（0.5x-2x）、音量、全屏、画中画；视频下方显示当前播放时间点 / 总时长
  - 阅读/课件类型：幻灯片内容区，支持翻页（上一页/下一页按钮或左右箭头键盘快捷键），显示当前页码 / 总页码；支持缩放（80%-150%）
- **底部标签页（4 个 Tab）**：
  - **课程介绍**：课程基本信息、课程简介、教师介绍、学习数据统计（本课程学习时长、完成度、测验平均分）
  - **资料下载**：配套 PDF/文档列表，每行显示文件名+大小+下载按钮；未登录或课程未开始时不显示
  - **学习笔记**：
    - 文本框记录笔记（富文本，支持加粗/列表/链接）
    - 「保存」按钮，保存后进入历史笔记列表
    - 历史笔记列表：时间倒序，支持编辑/删除/搜索
    - 笔记可设置"公开"（同课程学生可见）或"私有"
  - **讨论区**：
    - 本小节专属讨论帖列表
    - 发帖：标题+内容（富文本），支持 @ 同学
    - 回帖：嵌套层级展示，支持点赞、回复、举报
    - 教师回帖置顶标识
    - 热门帖优先展示

*3. 测验页*
- **头部信息**：测验标题、题量（X 题）、满分（X 分）、限时（X 分钟，0 为不限）、剩余时间倒计时（限时测验）
- **答题区**：
  - 单选题：选项垂直排列，点击选中（圆形单选框），提交后正确选项显示绿色 ✓，错误选项显示红色 ✗，正确答案高亮
  - 多选题：方形复选框，提交后部分正确显示黄色 ⚠️（得一半分），全对绿色，全错红色
  - 判断题："正确"/"错误"两个大按钮，提交后标记对错
  - 简答题：多行文本输入框，提交后显示参考答案（教师预设）
- **答题卡**：右侧悬浮（桌面端）或底部抽屉（移动端），显示所有题号，颜色标记（灰=未答 / 绿=已答 / 黄=标记），点击题号跳转
- **操作按钮**：标记本题（后续重点关注）、上一题/下一题、提交答卷
- **提交前校验**：若有未答题目，提示"还有 X 题未作答，确认提交？"（继续答题 / 确认提交）
- **提交后结果页**：
  - 得分（大数字）+ 正确率百分比
  - 答题卡（绿=对/红=错/黄=半对）
  - 逐题解析：每题展示正确答案+个人答案+解析+知识点标签
  - 操作：「重新答题」（若允许重考）/ 「查看答案」（若不允许重考但允许看答案）/ 「返回课程」

*4. 作业页*
- **头部**：作业标题 + 截止时间倒计时（若已过截止显示"已截止"红色标签）
- **作业要求区**：题目内容（富文本渲染）+ 作答要求 + 附件要求说明
- **作答区**：
  - 文本作答：多行富文本编辑器
  - 附件上传：拖拽或点击上传，支持多文件，显示文件名+大小+删除按钮；仅当作业要求附件时展示
- **提交按钮**：提交前二次确认"提交后不可修改，确认？"；截止后提交按钮禁用，显示"已截止，请联系教师"
- **提交后状态页**：
  - 「已提交」状态标签 + 提交时间
  - 展示提交内容（文本+附件，只读）
  - 等待批改提示
  - **作业互评**（教师开启后）：
    - 展示分配到的同伴作业（匿名，隐藏学生姓名）
    - 评分维度：教师预设（如"内容完整性"/"逻辑性"/"创新性"），每项 1-5 星
    - 文字评语：多行文本
    - 提交互评后展示对方给自己的评分（匿名）

#### 4.2.4 进度与完成规则

| 内容类型 | 完成标准 | 进度计算 |
|---------|---------|---------|
| 视频/阅读 | 进入即视为完成（演示阶段逻辑） | 完成节数 / 总节数 |
| 测验 | 提交后计为完成，得分计入学习数据 | 同上 |
| 作业 | 提交后计为完成 | 同上 |
| 章节 | 全部节完成 → 该章 100% | 已完成章节 / 总章节 |
| 课程 | 全部章节完成 → 课程 100% | 全局进度条 |

> 演示阶段视频/阅读"进入即完成"，后续可升级为"视频观看进度 ≥80% 或阅读停留时长 ≥ 预估时长"才计完成。

#### 【本节关键业务规则】

28. **学习状态持久化**：学生学习进度、笔记、讨论、测验记录实时保存，刷新页面后恢复至最近状态。
29. **测验防作弊**：限时测验倒计时后端同步，前端倒计时仅作展示；切换标签页/窗口超过 3 次自动提交；复制题目内容触发警告。
30. **作业截止控制**：截止时间精确到分钟，截止后禁止提交；教师可延长截止时间（通知已提交学生）。
31. **互评匿名性**：互评过程中学生姓名全程匿名，以"学号后四位+同学"代称；互评成绩按教师设定权重计入总成绩。
32. **笔记隐私**：私有笔记仅自己可见；公开笔记同课程学生可在讨论区侧边栏查看。

#### 【本节关键业务规则】

33. **成绩发布控制**：成绩未经教研组长审核不得发布；发布后学生方可查看；发布前学生端显示"成绩录入中"。

35. **排名可选展示**：教师/教研组长可配置是否向学生展示班级排名；默认不展示，保护学生隐私。
36. **申诉时效**：学生可在成绩发布后 7 个工作日内申请复核；超期后申诉入口关闭。

### 5.1 教学日程

#### 5.1.1 数据来源与展示

**数据来源**：教务系统每学期开学前推送本学期课表，包含：教师+班级+课程+上课时间+上课地点+课时序号。本平台只读展示，不支持在本平台排课、调课、代课。

**视图切换**：
- **周视图（默认）**：周一至周日横向时间轴，每天按节次纵向排列（如上午 1-2 节、3-4 节、下午 5-6 节等）
- **月视图**：日历格展示，有课日期标记品牌色圆点，点击日期展开当日课程列表
- **列表视图**：按时间倒序排列所有课程，适合查找历史课程

**导航**：上周 / 本周 / 下周 / 今日（快速回到当前周）

#### 5.1.2 课程卡片

**周视图卡片内容**：
- 课程名称（两行省略）
- 上课时间（如"第 3-4 节 10:00-11:40"）
- 上课地点（教室号）
- 班级名称（如"软件工程 2301 班"）
- 备课状态标签：
  - 未备课：红色标签 + 卡片边框红色细线
  - 备课中：黄色标签（已保存备课草稿但未完成）
  - 已备课：绿色标签
- **备课提醒**：上课前 48 小时内未备课课程，卡片顶部显示红色闪烁提醒条「距上课还有 X 小时，请尽快备课」

**卡片展开（点击展开箭头）**：
- 班级名单入口（查看学生列表，来自教务系统）
- 进入备课按钮（跳转备课工作台）
- 进入上课按钮（已备课状态展示，未备课时禁用并提示"请先完成备课"）
- 历史课堂记录入口（展示该课程历史上课次数）

#### 5.1.3 课表同步规则

- 教务系统调课/代课后，实时推送变更消息至本平台，对应课程卡片显示「课表已更新」提示，点击刷新
- 若教务系统删除某条排课，本平台标记为"已取消"（灰色置灰），保留历史记录但不可备课/上课

#### 【本节关键业务规则】

39. **课表只读**：本平台不排课、不调课、不代课；所有课表操作在教务系统完成，同步更新到本平台。
40. **备课强制**：未备课课程禁止进入上课；教师可跳过完整备课流程，但至少需确认课件资源（系统默认关联课程主讲课件）。
41. **课表失效处理**：教务系统推送"课程取消"后，本平台保留记录但禁用操作；已产生的课堂数据和成绩仍然保留。

### 5.2 备课工作台

#### 5.2.1 进入方式

- 从教学日程点击「进入备课」
- 从课程列表点击「备课」
- 首次进入时自动加载该课程最新已发布版本的内容作为备课素材

#### 5.2.2 三环节切换结构

顶部固定三环节 Tab：课前 → 课中 → 课后，带完成状态标记（未完成红点 / 已完成绿勾）

左侧固定信息栏：
- 排课信息卡片：日期 / 星期 / 班级 / 教室 / 课时序号 / 课程名称
- 课程目录树：来自课程管理平台的体系课目录，可点击预览节点内容（只读）

右侧活动流编辑区：
- 支持拖拽组装各类教学卡片
- 卡片可上下排序、删除、折叠/展开
- 自动保存（每 30 秒或切换环节时）

#### 5.2.3 课前环节

可添加的卡片类型：

| 卡片类型 | 配置内容 | 学生端展示 |
|---------|---------|-----------|
| **导学教案** | 富文本编辑器，教师编写本课时导学内容 | 上课前 24 小时自动推送至学生学习台"课前预习"区 |
| **学习目标** | 从课程管理平台同步，可补充本课时专属目标 | 同步展示 |
| **课前预习题** | 从题库选择题目（单选/多选/判断），配置限时 | 学生课前完成，上课后自动关闭；教师端展示完成率 |
| **预习资料** | 从资源中心选择文件，或上传新文件 | 学生可下载/预览 |

**课前完成标准**：至少配置 1 项内容（导学/预习题/资料），否则系统标记"备课中"而非"已备课"。

#### 5.2.4 课中环节

可添加的卡片类型：

| 卡片类型 | 配置内容 | 课堂执行方式 |
|---------|---------|-------------|
| **课件资源** | 从课程管理平台同步主讲课件，可选择性替换/补充 | 教师上课态主区域加载放映 |
| **课堂签到** | 选择签到方式：普通签到（一键）/ 限时签到（设定倒计时，如 3 分钟）/ 位置签到（基于 IP/地理位置，可设范围） | 上课后教师点击"开始签到"，学生端弹出签到面板 |
| **随堂测验** | 从试卷库选择试卷，或从题库快速组卷（配置题数/难度） | 教师点击"下发测验"，学生端实时收到；可配置限时 |
| **互动讨论** | 设置讨论主题、是否匿名、持续时间 | 教师点击"发起讨论"，学生端弹出讨论面板 |
| **课堂笔记** | 教师预设本课时重点笔记框架 | 学生可在学习页查看并补充 |

**课中完成标准**：必须配置课件资源（系统自动关联课程主讲课件作为默认，教师可确认或替换）。

#### 5.2.5 课后环节

可添加的卡片类型：

| 卡片类型 | 配置内容 | 执行时机 |
|---------|---------|---------|
| **课后作业** | 从作业模板选择，或新建；配置截止时间 | 下课后自动推送至学生学习台作业区 |
| **课后测验** | 从试卷库选择，配置开放时间（如下课后立即/24小时后） | 按配置时间自动开放 |
| **学习反馈问卷** | 从问卷模板选择（满意度/难点反馈/建议），或新建 | 课程结束后推送 |

**课后完成标准**：无强制要求，未配置课后任务标记为"已备课（无课后任务）"。

#### 5.2.6 备课状态流转

```
未备课 → 保存课前/课中任意内容 → 备课中
备课中 → 课中环节配置课件资源并保存 → 已备课
已备课 → 上课前仍可修改，保存后更新备课内容
已备课 → 上课后锁定备课内容（防止覆盖课堂记录）
```

#### 【本节关键业务规则】

42. **备课素材只读**：备课工作台中引用的课程内容（课件/知识点/测验）来自课程管理平台，只读展示，不可修改；如需修改需回到课程管理平台编辑并重新发布。
43. **备课自动保存**：备课内容每 30 秒自动保存，切换环节时强制保存；未保存离开提示确认。
44. **课后锁定**：课堂结束后（点击"下课"），该课时的备课内容自动锁定，不可修改，确保课堂记录与备课版本一致。
45. **备课复用**：同一课程多次上课（如每周一次），支持"复制上次备课"功能，复制后可在新课时基础上微调。

### 5.3 课堂上课

#### 5.3.1 进入上课

- 从教学日程点击「进入上课」（仅已备课课程展示此按钮）
- 或从备课工作台点击「开始上课」
- 进入后加载该课时的课件与备课配置

#### 5.3.2 课堂布局（教师端）

```
+------------------------------------------------------------------+
|  顶部栏：课程名称 + 班级 + 当前时间 + 上课时长计时器 + 下课按钮    |
+------------------+---------------------------+-------------------+
|   左侧课件列表    |        中间课件投屏区       |   右侧学生面板    |
|   （页码/节点）   |   （全屏/批注/聚焦模式）    |   （状态+答题）   |
+------------------+---------------------------+-------------------+
|            底部互动工具栏（固定悬浮）                              |
+------------------------------------------------------------------+
```

**顶部栏**：
- 课程名称 + 班级名称
- 实时时间
- 上课时长计时器（点击上课后自动计时，格式 HH:MM:SS）
- 「下课」按钮：红色，点击弹出确认"确认结束本节课？结束后将生成课堂记录并锁定备课。"（确认 / 取消）

**左侧课件列表**：
- 展示当前课时的所有课件页/节点，按顺序排列
- 每页缩略图 + 页码 + 节点名称
- 点击切换至对应页，学生端同步跟随
- 当前页高亮指示

**中间课件投屏区**：
- 主讲课件渲染区，支持 PPT/PDF/图片格式
- 控制栏：上一页 / 下一页（键盘左右键同步支持）/ 批注模式（画笔/荧光笔/橡皮擦，颜色选择）/ 聚焦模式（局部放大）/ 全屏放映
- 批注内容实时同步至学生端

**右侧学生面板**：
- **学生状态统计**：在线人数 / 专注人数 / 挂机人数 / 缺席人数（实时更新，每 10 秒刷新）
- **答题实时进度**（下发测验后展示）：
  - 答题人数 / 总人数 进度条
  - 平均得分（实时计算已提交学生）
  - 学生答题状态列表：已完成（绿）/ 答题中（黄）/ 未开始（灰）
  - 点击学生姓名可查看该生答题详情（仅教师可见）

**底部互动工具栏**（从左至右）：
1. **签到**：点击展开签到面板（开始签到/结束签到/查看实时结果）
2. **一键下发测验**：点击弹出备课时已配置的随堂测验列表，选择后点击"下发"，学生端实时收到并开始答题
3. **举手抢答**：点击"开始抢答"，学生端展示抢答按钮，第一个点击的学生头像展示在教师端"抢答成功"弹窗
4. **随机点名**：点击后系统随机抽取 1 名在线学生，展示头像+姓名+学号，学生端弹窗提示"你被点名了"
5. **发起讨论**：弹出备课时已配置的讨论主题，学生端进入讨论面板
6. **课堂笔记**：教师可实时编写课堂笔记，同步推送至学生端

#### 5.3.3 签到流程

**教师端**：
- 点击「签到」→ 选择签到方式（普通/限时/位置）→ 点击「开始签到」
- 限时签到：设定倒计时（1-10 分钟），学生端显示倒计时
- 位置签到：可设定允许签到范围（如"教学楼 A 座 500 米内"）
- 实时结果面板：展示已签到 / 未签到名单，手动修改状态（出勤/迟到/早退/请假/缺勤）并备注原因
- 点击「结束签到」关闭签到入口，已签到数据自动保存

**学生端（上课中）**：
- 签到开始后，学生端顶部弹出签到条，点击「签到」按钮
- 限时签到展示倒计时，超时后按钮禁用
- 位置签到需授权地理位置，超出范围提示"不在签到范围内"
- 签到成功后展示状态（出勤/迟到，根据时间自动判定）

#### 5.3.4 随堂测验流程

**教师端**：
- 点击「一键下发测验」→ 展示本课时备课配置的随堂测验列表 → 选择测验 → 点击「下发」
- 下发后学生端实时收到，右侧学生面板切换为"答题监控"模式
- 教师可实时查看：已提交人数、平均得分、每题正确率（柱状图）、常见错误选项（饼图）
- 点击「结束测验」→ 强制收卷（未提交学生自动提交）→ 展示全班成绩分布（柱状图）
- 可选择「讲解模式」：逐题展示正确率，重点讲解错误率高的题目

**学生端（上课中）**：
- 测验下发后，屏幕右侧弹出测验抽屉（不遮挡课件主区域）
- 答题界面同学生学习平台测验页（单选/多选/判断/简答）
- 提交后即时显示得分（若教师配置允许）
- 教师结束测验后，展示全班平均分和排名（若教师开启）

#### 5.3.5 下课与课堂记录生成

教师点击「下课」并确认后：
1. 自动保存所有课堂数据
2. 生成课堂记录（详见 5.4）
3. 锁定本课时备课内容（不可再修改）
4. 推送课后作业/测验至学生学习台（若备课时已配置）
5. 教师端展示"课堂总结"弹窗：
   - 本课时长
   - 出勤率
   - 互动统计（抢答人数/点名人数/测验平均分）
   - 快捷入口：查看完整课堂记录 / 进入课后备课（布置作业）

#### 【本节关键业务规则】

46. **上课准入**：仅"已备课"课程可进入上课；未备课点击上课按钮时拦截并引导至备课工作台。
47. **课堂实时同步**：教师翻页、批注、下发测验、发起互动后，学生端需在 1 秒内收到并响应；网络异常时学生端提示"连接中断，正在重连..."
48. **签到时效**：普通签到无时间限制，教师手动结束；限时签到超时后自动结束，未签到学生记为缺勤（可手动修改）。
49. **测验强制收卷**：教师点击"结束测验"后，未提交学生自动提交已答内容；简答题未答记 0 分，选择题未答不扣分但也不得分。
50. **课堂数据归属**：所有课堂互动数据（签到、测验、抢答、点名、讨论）归属该课时课堂记录，用于平时成绩计算。

### 5.4 课堂数据记录

#### 5.4.1 记录生成

每节课下课确认后，系统自动生成课堂记录，不可手动删除（仅管理员可归档）。

#### 5.4.2 记录内容

**基础信息**：
- 课程名称、班级、教师、日期、时间、课时序号、上课地点
- 课堂时长（实际计时）
- 关联备课版本号

**出勤数据**：
- 应到人数（班级总人数）
- 实到人数、出勤率
- 明细名单：学生姓名/学号/签到时间/签到方式/考勤状态（出勤/迟到/早退/请假/缺勤）
- 教师手动修改记录（修改人/修改时间/原因）

**互动数据**：
- 签到：发起次数、总参与人数
- 抢答：题目/参与人数/抢答成功学生/用时
- 随机点名：被点名学生名单/应答状态
- 随堂测验：测验名称/提交率/平均分/最高分/最低分/标准差
- 讨论：主题/参与人数/发帖数

**测验数据**：
- 每题正确率（柱状图）
- 常见错误选项分布（选择题）
- 学生个体答题详情（答对/答错/未答/用时）

#### 5.4.3 记录查看与导出

**教师查看**：
- 教学日程 → 历史课程卡片 → 查看课堂记录
- 列表展示所有历史课堂，支持按课程/班级/时间筛选
- 点击单条记录展开详情（上述所有数据）

**导出**：
- 「导出 Excel」按钮，包含：基础信息表 + 出勤明细表 + 测验成绩表 + 互动统计表
- 「导出 PDF 课堂报告」：排版打印版，含图表

#### 【本节关键业务规则】

51. **记录不可篡改**：课堂记录生成后，教师仅可补充备注或修改考勤状态（需留痕），不可删除记录或修改测验成绩。
52. **考勤数据保留**：课堂出勤数据保留一学期，学期结束后自动归档；归档后仅管理员可查看。
53. **成绩计算来源**：平时成绩中的"出勤率"和"课堂互动分"直接从课堂记录提取，自动按教师配置的权重计算。

### 5.5 学情分析

#### 5.5.1 班级学情看板

**入口**：智慧课堂平台 → 学情分析 → 班级学情

**筛选**：学期 / 课程 / 班级（三级级联，数据来自教务系统）

**核心指标卡片**：
- 班级平均出勤率
- 班级平均作业提交率
- 班级平均测验得分
- 班级平均学习进度（学生学习平台数据回传）

**图表区**：
- **成绩分布柱状图**：各分数段（90-100/80-89/70-79/60-69/<60）人数分布
- **知识点掌握热力图**：横轴为课程知识点，纵轴为学生列表，单元格颜色深浅表示掌握程度（基于测验正确率）
- **作业提交趋势折线图**：近 10 次作业的班级提交率曲线
- **出勤率趋势**：近 10 节课的出勤率曲线

**学生排名列表（教师/教研组长可见，不展示给学生）**：
- 列：学生姓名、学号、学习时长、作业平均分、测验平均分、出勤率、总评、操作
- 操作：查看个体详情 / 发送提醒（站内消息）
- 支持按任意指标排序

#### 5.5.2 学生个体分析

**入口**：班级学情看板点击学生姓名，或搜索学号/姓名

**学生画像卡片**：
- 基本信息 + 本学期课程数 + 总学习时长
- **学习时长趋势**：近 30 天每日学习时长折线图
- **成绩变化曲线**：本学期各课程总评成绩变化
- **薄弱知识点列表**：按测验错误率倒序排列，展示知识点名称、错误次数、关联课程节点
- **课堂表现**：出勤率、抢答参与次数、点名应答率、随堂测验平均分

**教师操作**：
- 「发送学习提醒」：选择预设模板（如"最近学习进度较慢，请加快学习节奏"）或自定义消息，推送至学生学习台消息中心
- 「标记重点关注」：标记后该学生在本系学情看板中置顶展示

#### 5.5.3 教学资源使用分析

- **视频观看完成率**：各课程视频的平均观看完成百分比
- **课件下载次数**：各资源被下载次数排名
- **作业提交率**：各次作业的提交率对比
- **测验难度分析**：各次测验的平均分、标准差、区分度（高分组与低分组正确率差）

#### 【本节关键业务规则】

54. **数据权限**：教师只能查看自己所授课程的班级/学生学情；教研组长可查看本系全部课程学情；管理员可查看全局学情。
55. **学情数据时效**：课堂数据实时更新；学生学习平台行为数据（学习时长/进度）T+1 同步至智慧课堂平台。
56. **排名隐私保护**：学情分析中的学生排名、成绩分布仅教师/教研组长可见，不开放给学生；学生个体仅在学习档案中查看自己的数据。
57. **提醒限制**：教师每周最多向同一学生发送 3 条学习提醒，防止骚扰。

## 6. 与教务系统对接

### 6.1 教务系统 → 本平台

| 数据 | 说明 | 同步时机 | 同步方式 |
|------|------|---------|---------|
| 学期课表 | 教师+班级+课程+上课时间+上课地点+课时序号 | 每学期开学前 | 批量推送 + 增量更新 |
| 学生名单 | 班级学生基本信息（姓名、学号、院系、专业） | 每学期开学前 | 批量推送 |
| 课程基础信息 | 课程名称、编码、学时、学分、课程属性 | 课程申报审批后 | 实时推送 |
| 实践成绩 | 场景化平台汇集后的学生实践成绩等级 | 实践环节结束后 | 批量推送 |
| 院系/专业枚举 | 组织架构与专业目录 | 有变更时 | 实时推送 |

### 6.2 本平台 → 教务系统

| 数据 | 说明 | 同步时机 | 同步方式 |
|------|------|---------|---------|
| 过程性成绩明细 | 平时成绩拆解（出勤率、作业均分、测验均分、互动分） | 实时或每日批量（教师提交成绩册后） | API 推送 |
| 期末总评成绩 | 总评分数、等级、绩点 | 学期末成绩发布后 | API 推送 |
| 出勤数据 | 每节课出勤记录（应到/实到/迟到/缺席名单） | 每节课结束后 | 实时推送 |
| 课堂互动数据 | 随堂测验成绩、抢答记录、点名应答记录 | 每节课结束后 | 实时推送 |
| 成绩修改记录 | 修改前值、修改后值、修改人、时间、原因 | 修改审批通过后 | 实时推送 |

### 6.3 数据一致性保障

- **推送失败重试**：失败时进入重试队列，间隔 5 分钟重试，最多 5 次；仍失败则通知管理员并记录异常日志。
- **数据校验**：接收教务系统数据时校验必填字段完整性，缺失字段标记"待补全"并通知管理员。
- **双向对账**：每学期末执行成绩数据对账，本平台推送成绩 vs 教务系统接收成绩，差异报告自动发送至管理员和教研组长。

## 7. 核心业务实体

**课程（Course）**
- 类型：体系课 / 颗粒课
- 属性：公共基础课 / 专业基础课 / 专业核心课 / 素质拓展课
- 状态：草稿 / 审批中 / 已发布 / 已驳回
- 基本信息：名称、编码、行业、专业、教师、简介、封面、学时、版本号
- 体系课关联批次分组 ID，颗粒课独立管理
- 包含审批记录、测验配置、作业配置

**体系课节点（CourseNode）**
- 层级结构：parentId 关联，最多三级
- 类型：原创 / 克隆 / 引用 / 资源节点
- 内容：学习目标、知识点标签、预估课时、资源列表、测验配置、作业配置
- 引用节点关联源颗粒课 ID，随源内容同步
- 资源节点仅含名称与资源文件
- 显示顺序字段控制同级排序

**颗粒课（GranularCourse）**
- 同课程基础信息，额外包含课程类型（知识/技能/素养）
- 自包含全部教学内容：学习目标、知识点、导学教案、资源、测验、作业
- 可被体系课节点引用或克隆

**题目（QuizQuestion）**
- 类型：单选 / 多选 / 判断 / 简答
- 题干、选项、正确答案、解析、分值、难度、知识点标签
- 归属题库（我的 / 共建 / 公共）

**试卷（ExamPaper）**
- 题目列表（含顺序与分值分配）、总分
- 考卷设置：时长、重考、乱序、成绩显示、启用条件
- 组卷方式：手动 / 自动

**作业（Homework）**
- 题目、要求、是否需要附件、截止时间
- 关联课程或节点
- 互评配置：是否开启、互评份数、互评权重

**课堂记录（ClassRecord）**
- 关联排课记录（来自教务系统）、教师、班级、时间、地点
- 出勤数据：应到/实到/迟到/缺席名单与明细
- 互动数据：签到/抢答/点名/讨论参与统计
- 随堂测验数据：提交率/平均分/每题正确率/学生个体答题详情
- 关联备课版本号

**成绩记录（GradeRecord）**
- 学生、课程、学期
- 成绩项：平时得分、期中得分、期末得分、实践得分、总评成绩、等级、绩点
- 平时成绩拆解：出勤率、作业均分、测验均分、互动分
- 录入人、审核人、发布时间
- 修改记录：修改前值、修改后值、修改人、时间、原因

**学习记录（LearningRecord）**
- 学生、课程、节点
- 进度状态：未开始/进行中/已完成
- 学习时长、视频观看进度、最近学习时间
- 测验/作业提交记录与得分

**审批记录（ApprovalRecord）**
- 关联资源（课程/成绩册）、审批人
- 操作：通过 / 驳回 / 修改批准
- 备注/原因、时间
- 永久留存，不可删除

**教学日程/排课记录（ClassSchedule）**
- 关联教师、课程、班级、教室
- 日期、时间、课时序号
- 备课状态：未备课 / 备课中 / 已备课
- 数据来源：教务系统推送，本平台只读

**考勤记录（AttendanceRecord）**
- 关联课堂记录、学生
- 签到方式、签到时间、考勤状态（出勤/迟到/早退/请假/缺勤）
- 教师手动修改记录：修改前状态、修改后状态、修改人、时间、原因

**备课记录（PrepRecord）**
- 关联排课记录、课程版本
- 课前内容：导学教案、预习题、预习资料
- 课中内容：课件资源确认、签到配置、随堂测验配置、互动讨论配置
- 课后内容：作业配置、课后测验配置、反馈问卷配置
- 状态：备课中 / 已备课 / 已锁定（上课后）

**知识图谱节点（KnowledgeGraphNode）**
- 名称、学科分类、难度等级
- 关系：前置知识 / 后置知识 / 关联知识（关联表存储）
- 与课程知识点手动映射

**资源文件（ResourceFile）**
- 文件名、类型、大小、分类、存储路径、上传人、上传时间
- 引用次数统计
- 引用链记录（被哪些课程/节点引用）
